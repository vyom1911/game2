The files required to run this game are project1.html and project1.js. A web browser is required to run the game.
Open the project1.html in the web browser to run the game.
The project also requires an images folder which includes image files and spritesheets.

The following files and folder should be under the same directory:
Images (folder)
project1.html (file)
project1.js (file)

The images folder will include:
Arrow.png
Grass.png
hero.png
zombie.png
