About the Game:

The player is in a zombie Apocalypse and has to survive for 200 seconds. The player has a bow and arrow to kill the zombies and the number of zombies the player kills is his score. The player can shoot the arrow from bow and can control the arrow after it is released from bow. When one zombie dies 2 more are generated and so on.

Instructions:

  • Use the arrow keys to move the player
  • Use the spacebar to shoot the arrow
  • Control the arrow using the arrow keys
